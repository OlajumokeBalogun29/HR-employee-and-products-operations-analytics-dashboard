# HR Employee & Product Analytics Dashboard

## 📊 Project Overview
A Power BI dashboard analysing workforce demographics, 
compensation, sales performance & product inventory across 
HR and Operations. Covers 290 employees, 16 departments, 
$23.69M in sales & 504 products.

## 🔍 Key Insights
- 71% male vs 29% female workforce — diversity gap identified
- Hiring peaked in 2009 — critical talent pipeline gap
- 55-point pay rate spread across departments
- BB Ball Bearing dominates inventory at 0.91M units
- Fork End is highest scrap-generating component

- DAX
- Power Query
- Microsoft Excel
